WatchTower

Personal Alerts & Monitoring System

WatchTower is a personal software project that fetches data from various sources such as weather, stock prices, or other alerts you specify, and sends notifications via email using Brevo API. This project runs autonomously using Python scripts and GitHub Actions.

The images above change based on the current Philippine time to reflect day, evening, or night.
